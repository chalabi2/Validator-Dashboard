# Validator Dashboard

An easy to use dashboard for everyday validator actions.

## Instructions

### Run locally
```
cd Validator-Dashboard
npm install
npm start
```
Open [http://localhost:1234](http://localhost:1234).

### Build for deploying 

After building, you can serve the `dist` folder

```
cd package_code
npm install
npm run build
serve -s dist
```
